# Astro Chat

## Overview
The Astro Chat project features a dynamic landing page that leverages cutting-edge technologies to provide an engaging user experience. This README documents the implementation details of the landing page, including UTM personalization, language switching, campaign messaging, chat SDK integration, and event tracking.

## Features

### 1. Dynamic Landing Page Implementation
- A responsive landing page built with modern web technologies (HTML5, CSS3, JavaScript).
- Utilizes a modular design approach to ensure easy maintenance and updates.

### 2. UTM Personalization
- The landing page captures UTM parameters from the URL to personalize user experiences.
- Different content can be displayed based on campaign source, medium, and other parameters.

### 3. Language Switching
- Multi-language support allowing users to switch between different languages seamlessly.
- Language preferences are stored in the user’s local storage for persistence.
- Implemented using a simple dropdown selector.

### 4. Campaign Messaging
- Ability to display tailored messages based on the marketing campaign.
- Custom messages that can be configured dynamically via the backend.
  
### 5. Chat SDK Integration
- Integrated with a chat SDK to facilitate real-time communication.
- Users can engage with support or conversational agents directly from the landing page.
- Includes features such as message history, typing indicators, and notifications.

### 6. Tracking Events
- Comprehensive event tracking implemented to measure user interactions on the landing page.
- Events tracked include user clicks, language switches, and chat initiations via Google Analytics/Tag Manager.
